MP-CU文档，ColorUI 微信原生小程序UI组件库

开始使用 GitHub open in new window Gitee open in new window

组件

ColorUI3.x 对比 ColorUI2.x，从之前的class方式，转为了组件化的方式，并提供了丰富的自定义参数以及插槽

颜色

漂亮及丰富的色彩，一直是ColorUI的亮点，ColorUI3.x内置了比ColorUI2.x更多的预设颜色（普通色、渐变色等）

主题

内置了两种主题，白色和深色，在开发使用时，只需要简单的调用，即可在微信原生小程序上拥有全局主题功能
